반복문 : 연속으로 횟수만큼 반복처리한다.
        for(초기값; 최종값; 증감식){
            실행문
        }
        초기값: 시작값, i , j , k.. 순으로 작성
        최종값: ~까지 조건식 true/false 
        증감식: ++, --, 수식

        반복 : for / while 을 많이쓴다. 
        
        1. for : 횟수가 정해져있을때(무한반복된다)
            for(초기값; 최종값; 증감식){}
            for(;;){} : 무한반복(잘안씀)

        2. while : 무한반복일때더 쓴다.
            초기값
            while(조건식){
                실행문
                증감식
            }

            while(true){
                무한반복
            }

        3. do ~while : 무조건 한번은 실행
